# 🚀 LinkedDev-AI

Transform GitHub repositories, technical blogs, and documents into polished, developer-focused LinkedIn posts in seconds!

## ✨ Features

- 🔗 **Multi-Source Input:** Import content from GitHub repositories, articles, or Markdown/text files.
- 🔍 **Smart Extraction:** Uses BeautifulSoup4 to gather relevant information from webpages.
- 🧩 **RAG-Based Chunking:** Employs `RecursiveCharacterTextSplitter` to preserve context in long documents.
- 🤖 **AI-Powered Generation:** Utilizes Google Gemini (`gemini-2.5-flash`) to craft professional LinkedIn posts.
- 🎨 **Beautiful UI:** Streamlit interface with customizable Dark and Light themes.
- ⏳ **Reliable Experience:** Automatic retries gracefully handle temporary API rate limits.
- 🔗 **Developer-Centric Output:** Produces concise, authentic posts with links back to your projects.

## 🛠️ Tech Stack

- Python
- Streamlit
- BeautifulSoup4
- LangChain Text Splitters
- Google GenAI SDK (Gemini 2.5 Flash)

## 💡 Why LinkedDev-AI?

Developers should spend more time building and less time worrying about how to present their work. LinkedDev-AI streamlines project sharing while keeping the spotlight on the engineering itself.

## 🚀 Getting Started

1. Clone the repository.
2. Install dependencies.
3. Add your API key via environment variables (never commit secrets).
4. Launch the Streamlit app and start generating posts!
